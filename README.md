# Retail Customer Segmentation & Loyalty Database

## Project Scope
For this project, I plan on creating a retail CRM (Customer Relationship Management) database that stores data ranging from customers, products, and placed orders. This database is being created to precisely observe behaviors within customer spending, purchased products, and retention. This project will include SQL queries that will calculate a "Customer Lifetime Value", and properly catergorize customers into categories such as high-value or frequent buyers.

## Users
This databaes is intended to to allow retail business owners, CRM analysts, and marketing teams to accurately observe certain trends and loyal customer patterns. This information will assist them with creating customer promotions, loyalty programs, and retention strategies.

## Data Sources
The data used for this project is sourced from a public "Customer Personality Analysis" dataset found on kaggle.com. If needed, synthetic retail data will be generated to allow for proper CLV calculations and customer segmentation.

## Database Application
For this project, I plan to implement my database system using the Oracle Database through the Oracle SQL Developer desktop application. I plan to use Oracle SQL Developer to create and enforce all of the relationships within the tables, primary keys, foreign keys, and constraints.
